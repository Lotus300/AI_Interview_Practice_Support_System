# AI面接練習支援システム デザインデータ

## 採用ステータス

このデザインを画面設計の採用版とする。

- モック: `outputs/screen-mock/index.html`
- TypeScriptソース: `work/screen-mock/src/app.ts`
- スタイル: `outputs/screen-mock/style.css`
- デザインデータJSON: `outputs/screen-mock/design-data.json`

## デザイン方針

- 個人利用の面接練習に集中できる静かな業務アプリ調デザイン
- ハンバーガーメニューによるドロワーナビゲーション
- 面接実施画面では音声入力、音声認識、LLM分析、VOICEVOX再生の状態を明示
- 音声・LLM処理に失敗しても、再試行またはテキスト入力で継続可能
- 矛盾点は断定せず、確認候補として表示

## コンポーネント

| コンポーネント | 内容 |
|---|---|
| ハンバーガーメニュー | 画面一覧をドロワーで表示。画面選択または背景クリックで閉じる |
| ステータスバッジ | `blue`, `green`, `yellow`, `red` の状態表示 |
| 入力項目 | ラベル上部、入力下部の縦積み |
| 数値コントロール | 話速・音量を `0.1` 刻みで調整。スライダー、`- / +`、長押し対応 |
| モック用状態切替 | 面接実施画面の状態確認専用。実画面の通常操作ではない |

## 採用画面

| 画面ID | 画面名 | 主な役割 |

| SCR-008/009 | 履歴画面 | 過去の練習結果確認 |
| SCR-010/011 | 設定画面 | プロフィール・音声設定 |
| SCR-012 | エラー・復旧画面 | 失敗時の継続導線 |
